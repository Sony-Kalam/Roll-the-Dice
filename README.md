# Roll The Dice 🎲

A beginner-friendly Java program that simulates a two-player dice game. Each player rolls a die, and the higher number wins the round. The game continues until players choose to stop.

## How to Play
1. Run the program in your terminal.
2. Player 1 rolls the die.
3. Player 2 rolls the die.
4. The program announces the winner of the round.
5. Choose whether to play again or exit.

## Features
- Random dice rolls using `java.util.Random`
- Two-player competition
- Replay option for multiple rounds
- Simple console-based interaction
